新坑预告：基于 Flask 与 React 的开源实用工具集 Applied Toolkit
========================================================================

动机
--------

　　编程(Programming)，或曰规划，其本意是指设计一个符合逻辑的可控体系。这一行为最令人着迷的特点之一便是，在计算机的帮助下，我们编写出的程序可以按照代码所声明的控制流自动而正确的执行（虽然实践中并不总是如此）。编程的意图可以看作是智能体对自身作用于世界的影响力进行拓展——以智能体的逻辑思维过程为载体的决策与信息处理过程可以发生在智能体自身之外，乃至完全脱离智能体自身而作用于世界。于个人而言，编程极大地扩展了个体对于世界的影响力也即改造世界的能力，进而增强了个体的理解能力、分析能力与判断能力。

　　而在编程这一实践的诸多作用之中，最基本也最常见的便是对信息处理效率的提升。因此，我计划着手编写一系列能够提升日常生活中某个特定行为效率的组件，并尝试着将这些组件整合到一个通用的架构之中，于是便有了这一个被称作“实用工具集”(Applied Toolkit)的项目。

规划
--------

　　作为一个影视作品与电子书籍爱好收藏者，我苦于缺少一个美观、功能丰富且易于上手的通用内容管理与检索软件，因此本项目的第一阶段开发工作将主要聚焦于这一部分。其他已纳入考虑的组件主要包括：可视化的工作流调度管理器、Minecraft 地图查看器、机器学习实验管理器等等。

造轮子？
--------

　　1. 解决特殊需求
　　2. 锻炼架构/抽象/开发能力
　　3. 前后端协作开发实践
